Components

Checkboxes

Checkboxes signal an action that will let the user navigate to another view, when they click or tap on them.

Using checkboxes

Checkboxes must feature a descriptive label.

Designing

Checkbox
Checkboxes make the selection visually distinct.

Checkbox label
Icons are used to simplify user recognition and provide ability to shorten link text to a minimum. Icons should be limited to differentiate from a standard hyperlink (i.e. if it does something other than navigating to a new page).
